Description¶

The UNICORN Binance Suite is a Python Meta Package of

  • unicorn-fy

  • unicorn-binance-rest-api

  • unicorn-binance-websocket-api

Coming soon¶

  • unicorn-trailing-stop-loss-engine

  • unicorn-local-orderbook-manager

Installation and Upgrade¶
